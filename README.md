➕🔃💸 Emo 🤐☀💵
======

Emo suggests you emoji. ☂🐺😕🐠🕗



🗯🔎📜 Installation ⚜🗝🤑
--------


```bash
$ go get github.com/pocke/emo
```



🚽💊🚕 Usage 🎧🕧🚊
=========

```bash
Usage: emo [all|rand]

Display all emoji
$ emo all

Display an emoji randamly
$ emo rand

Display two emoji randamly
$ emo rand 2

Display many emoji randamly
$ emo rand 1000
```
